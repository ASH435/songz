# Spotify Playlist Player

A modern, responsive HTML/CSS/JS song player that displays tracks from a specified Spotify playlist with a glassmorphism design.

## Features
- **Modern UI**: Dark theme with glassmorphism effects (backdrop-filter).
- **Tracklist**: Searchable list of all 30 tracks from the playlist.
- **Dynamic Selection**: Selecting a track updates the "Now Playing" display.
- **Spotify Integration**: Integrated Spotify Embed for audio playback.
- **Responsive**: Works on both desktop and mobile devices.

## How to use
1. Clone the repository.
2. Open `index.html` in any web browser.
3. Use the sidebar to search for and select your favorite tracks.
4. Play the music directly through the Spotify Embed.
